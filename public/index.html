Plantimator

    Plantimator is my private set of tools for generating animated floral scenes. These can be used as page borders, page backgrounds, text borders, text underlines, and for generally decorative purposes.

    Also, Plantimator currently only supports Desktop Chrome.

    Tech used to build this site includes VS Code, Vanilla JS, Javascript, Firebase.

  • Plant Editor

    Used to define scenes or individual plants. Individual plants are composed of a stem, leaves, and flowers. Floral scenes consisting of any number of predefined plant animations.

  • Leaf Editor

    Used to define free form shapes that can be used as leaves and flowers.

  • Library

    List of leaf and flower patterns that can be used when compositing a plant.
